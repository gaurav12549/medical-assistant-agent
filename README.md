# AI-Powered Medical Assistant

This project is an AI-powered medical assistant that:
- Collects user symptoms
- Performs triage
- Suggests possible conditions
- Recommends basic care or tests
- Advises when to consult a doctor

## Tech Stack
- Python 3
- Streamlit (for web UI)
- Scikit-learn (for ML, if needed)
- Rule-based logic for triage

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Start the app: `streamlit run app.py`
